# Assignment: Create a Profile Card Using the CSS Box Model

## Overview
You will create a simple *profile card* on a webpage and style it using the CSS **box model**. Your work should clearly demonstrate how **content**, **padding**, **border**, and **margin** shape the layout of an element.

## What You Will Create
A small profile card containing:
- A heading (name)
- A short bio paragraph
- A styled box with padding, border, and margin
- *(Optional)* A second card to compare different box model styles

## Step 1: Write the HTML
Create a simple structure for your card:

```html
<div class="card">
  <h2>Jane Student</h2>
  <p>Future web developer. Loves pizza, CSS, and long walks through the DOM.</p>
</div>
```

## Step 2: Add CSS Demonstrating the Box Model
Your CSS **must include**:
- Padding
- A border
- Margin
- A set width
- Comments explaining why you chose these values

Example (customize this for your own design):

```css
.card {
  width: 300px;
  padding: 20px; /* space inside the border */
  border: 3px solid #333; /* the visible frame */
  margin: 30px auto; /* space outside the element */
  background-color: #f2f2f2;
}
```
